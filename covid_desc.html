About Covid App

Welcome to the Covid App about page where you can find information about the APP and API behind the application.

API Access

The data used for this application was accessed through the 'Promise.all()' approach that allowed use to make multiple fetch requests to both the country summary api and the daily summary api for different countries. This then allowed us to access the data from both the API's in an array where each api response is a json object. We retrieved the values for the country from the users through the use of the input box and the 'value' property of that input box that was accessed once the button was clicked then passed that as a variable to be used in the fetch requests to get the country that users searched for.

Data Content

The data we retrieved from the daily summary API had information on the daily cases, daily deaths and daily recoveries for each country. The country summary API had information on the total cases, total deaths and total recoveries for each country. We decided to focus on the daily cases and total cases for each country in the visualisation of this data.

Execution Visualisation

In order to execute this we used chart.js to create a visualisation of the data. We used the daily case for each country to create a line chart where the dates were used as labels. In order to achieve this we had to use the 'map' function to iterate through the array of json objects in order to get the 'date' and 'dailyConfirmed' properties to create the labels and the data that would be entered into the chart.
